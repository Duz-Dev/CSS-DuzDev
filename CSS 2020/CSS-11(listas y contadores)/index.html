11.1 Estructura de listas

Definición:

Las listas son una enumeración de elementos. En html hay dos tipos de listas, ul para las no ordenadas, y las ordenadas ol; y cada elemento de la lista es un li (list item). El navegador por default a las listas ul les pone balitas, y las ol números, esto lo dibuja el navegador.
Dentro de una lista podemos poner otras listas exactamente dentro de los li.

Esta seria una lista desordenada (ul) que dentro de uno de los li tiene otra lista que en este caso seria ordenada (ol) ○ Hola mundo ○ Hola mundo ○ Hola mundo 1. item1 2. item2 3. item3 4. item4 ○ Hola mundo ○ Hola mundo

11.2 List style type

Definición

Con la propiedad list style type podemos cambiar el estilo de la lista (cambiar la viñeta que muestra el navegador). Esta se la podemos aplicar ya se a el ul, ol o al mismo li, ya que en cualquiera de los casos que remos que el li adquiera dicha propiedad (lo heredara automáticamente).

list-style-type:;

también podemos aplicar diversas diversos valores a la propiedad ya que son muchas las variantes.

11.3 List style image

Definición:

Esta propiedad sirve para colocar una imagen en ves de una viñeta, la desventaja es que no es posible re-dimensionar dicha imagen ._., la única solución es colocando una imagen del tamaño adecuado para la lista. Un método es usar los seudoelementos con la propiedad background-img.

/*Ejemplo con ::before y bg-img*/ .box-sml-06 ol { list-style-type: none; } .box-sml-06 ol li::before { content: ''; display: inline-block; background-image: url("../media/images (3).jpg"); background-size: contain; width: 50px; height: 50px; } /*esta solo es una prueba con before, es mucho mejor que usar la misma list-style-image ._.*/

11.5 Contadores

Definición:

Con los contadores podemos contar todos los elementos que estan en una misma familia de clase, por lo que podemos contar cualquier cosa de html con css.

Para ello tenemos que crear un contador que es como una variable que se va incrementando, esta la creamos en el lugar donde vamos a trabajar(ya sea en el body o en el papá de los elementos que queremos contar), el contador es con la función couter-reset.

Couter-reset tiene dos funciones, crea un contador y también lo resetea ._.XD. A este le asignamos un nombre (Cual quier nombre, es indiferente cual escojamos), que en nuestro caso sera i.

Después de ello dentro del bloque de declaración de la clase o del objeto que deseamos contar en css escribimos counter-increment junto con el nombre de nuestra variable llamada i y seguidamente de esta el valor que se le asignara a todos lo objetos o clases, por ejemplo 1.

counter-increment: i 1; <-- uno es el valor por default de counter-increment -->

Ya solo quedaría imprimir nuestro resultado, y lo haremos por medio de la propiedad content de los pseudoelementos.

Selector::before { content: counter(i) '.- '; color: red; }

11.7 Ejemplos de contadores

Definición:

Creamos una caja donde estará una lista que mostrara los temas con sus subtemas, para diferenciarlos ponemos el numero de tema junto con el numero del subtema.

llamamos a la caja papá temario, y dentro de este ponemos otras 3 cajas que tendran un tema y sus respectivos subtemas; así que escribimos h3 con la clase clase-title y los subtitulos con la clase class-topic.

< div class="temario"> < div class="class" > < h3 class="class-title">Box model< /h3> < p class="class-topic">Width< /p> < p class="class-topic">height< /p> < p class="class-topic">padding< /p> < p class="class-topic">margin< /p> < /div> < /div>
.temario { counter-reset: title; } .class-title { counter-increment: title; margin-bottom: calc(var(--space) + 3px); } .class-title::before { content: counter(title) '. '; } .class { counter-reset: class; } .class-topic { counter-increment: class; margin-bottom: var(--space); margin-left: var(--space); } .class-topic::before { content: counter(title) '.' counter(class) '.
