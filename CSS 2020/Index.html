elementos bloque

box

Soy el titulo del box

Hola, soy el texto que va dentro del bloque box.

Puedes usar el bloque box dentro de los demás elementos ya que su alto es del 100% del contenedor padre y el alto es automático.

Estos si o si se tienen que escribir dentro de un container, y si queremos que sigan el flojo alineado de la pagina estos tendrán que estar dentro de un content.

Titulo del card

Soy un párrafo que esta adentro card.

Este bloque esta hecho por si se quiere adjuntar una descripción mas exacta sobre un elemento para ponerle una imagen como tal.

Esta se coloca obligatoria-mente dentro de un container y se puede colocar igualmente dentro de el bloque box.

Consola

console

Este es el bloque que utilizo para mostrar el código CSS aplicado a ciertos elementos para mostrar explícitamente que es lo que aplique exactamente. Este bloque tiene la característica que respeta los saltos de linea lo que significa que con tan solo copiar y pegar el código ya se ve ordenado.

Para complementar este mismo utilizo etiquetas span con clases que cambian el color para resaltar los colores de cada palabra como si estuviera escrita en sublime.

Las listas son importantes para enumerar o ordenar diferentes conceptos lo cual a la hora de organizar la información resultan muy útiles. Para ello cree un diseño de lista el cual se compone de la siguiente manera:

  • Soy un li
  • soy un li con la etiqueta code.
  • Soy un li

    con la etiqueta code y con un párrafo.

Evidentemente cambia mas a la hora del código ya que la manera en que esta escrito sigue un orden de tal manera que pueda interactuar con cada elemento por separado y no por cascada (aunque a veces esto sera lo mejor).

Fondos

Columnas relative(rer y rel)

Soy la columna rel

Estas culumnas funcionan así:

La columna con la clase col--rel es la columna que va orientada a la izquierda principalmente, por ello el nombre (rel significa relative + left) ya que esta mide mide casi el doble que si hermano rer.

Soy la columna rer

La columna con la clase col--rer es la columna que va orientada a la derecha gracias a que se calcula la suma de los anchos de ambas columnas menos el 100% lo que da un margin-left que es utilizado para empujar la 2da columna (indiferentemente de que sea rer o rel) solo usando float.

No importa el orden

Puedes utilizar la columna rer o rel en el orden que ti quieras, da igual cual vaya en que dirección, solo ten en cuenta que rer mide casi la mitad que rel.

Soy la columna rel

  • col--re

    es la clase que engloba esta dos columnas, podemos utilizarla para asignarle valores a ambas con una sola class.

  • col--rel

    es la clase que especifica un ancho con la variable --col-rel-width que mide casi el doble que si hermano rer.

  • col--rer

    es la clase que especifica un ancho con la variable --col-rer-width el cual casi mide la mitad que si hermano rel.
