10.4 Before y after

Definición:

Son similares a first-line y a first-letter, pero, before y after son dos elementos que no existen pero que son hijos del elemento; before seria el hijo al inicio del elemento y after seria el hijo al final del elemento. Con first ya apuntabas a el contenido como lo son las letras, pero con before y after no apuntas a ningún contenido, por lo tanto tienes que generarlo, ya que si no hay tal contenido no cambiara nada. Con esto usaremos CSS para generar contenido. A diferencia de html y js css no genera un contenido tal cual, pero si lo que necesitamos para nuestra tarea de hoy.
Usaremos la función 'Content()' de CSS Solo hay dos puntos importantes: el primero que para manipular su contenido siempre tiene que llevar el content aunque sea vacío, y el 2do es que el elemento sera inline así que para manipularlo tienes que ponerlo en block u otro.

10.10 Ejercicio: aspect radio

Definición:

Con aspect radio significa el aspecto, y en css no tenemos una forma de calcular el aspecto; ejemplo que una caja siempre se vea de 3 x 3 no hay forma de que se mantenga igual, lo mas cercano es con viewport pero no es igual. Pero hay hacks que igual ayudan a solucionar esto:
-uno seria con solo usar el padding-bottom con un 100% ya que este es relativo al papá para obtener un Cuadrado, pero si al hijo le establecemos un ancho este se se deformara y se convertirá en un rectángulo. -otra opción y que tiene mas ventajas es usar ya sea before o after para plasmar el padding y con la caja que se convertirá ahora en papá ya solo le asignaríamos el ancho que queramos.
