Object y filter

Object-fit

Propiedad:

Esta propiedad de css sirve para especificar como debe de verse una imagen o video para que se ajuste a su contenedor, por dar un ejemplo este es similar a background-size ya que tienen casi los mismos valores y efectos.

Sintaxis

Se escribe comoobject-fity esta se puede aplicar solo a lasimgo videos.
Sus valores son los siguientes:

  • fill

    Valor por default

  • contain

    El contenido se ajustará hasta rellenar de forma horizontal o vertical el contenedor sin deformarse

  • cover

    El contenido se ajustará completamente de forma horizontal y vertical el contenedor pero no podemos ver el objeto completamente.

  • none

    El contenido no se re-dimensiona y mantiene su tamaño original mostrando solo el trozo de las dimensiones especificadas, pero esto provocara que la imagen o video se deformen.

  • scale-down

    Selecciona el menor de la comparación entre none y contain, vaya hace una comparación entre si none mide tal y contain y el que de el un numero menor lo mostrara.

Object-position

Esta propiedad va de la mano de object-fit ya que sirve para cambiar la parte de la imagen que podemos visualizar cuando esta no se muestra por completo, por lo que solo la podemos usar si tenemos object-fit con cover o none.

La sintaxis es object-position:(primer valor en el eje x) (2do valor en el eje y);

Podemos usar los valores de left y right para el eje X y para Y top y bottom, y al igual podemos utilizar medias como px o porcentajes.

Ejemplo

En este ejemplo le he puesto object-fit:none;yobject-position:left top; que al pasar el ratón se movera a abajo a la derecha.

Filter

¿Qué es?

Como dice la propiedad este es un filtro que dota de efectos gráficos como el desenfoque o cambio de color en la representación antes de que se muestre el elemento. Los filtros se utilizan comúnmente para ajustar la representación de imágenes, fondos o bordes yq que estos denotan más.

Sintaxis

Se escribe tal cualfiltery le podemos asignar las siguientes propiedades:

  • none

    Este elimina todo efecto aplicado.

  • blur(px)

    desenfoca la imagen

  • brightness(%)

    Ajusta el brillo de la imagen siendo 1 el original. De 0 a 1 dan oscuridad y de 1 a n dan sobre exposición

  • contrast(%)

    Ajusta el contraste de la imagen siendo 1 el original. De 0 es negro y de 1 a n dan más contraste

  • drop-shadow(h-shadow v-shadow blur spread color)

    Aplica una sombra paralela a la imagen.

  • grayscale(%)

    Convierte la imagen a escala de grises, 0 es el original y 1 sería blanco y negro completamente

  • hue-rotate(deg)

    Añade matiz de color a la imagen. Se da un valor en grados según la rueda cromática. El valor máximo es 360deg.

  • invert(%)

    Invierte el color de la imagen. Saca un negativo. 0 es el valor por defecto y 1 es totalmente invertida

  • opacity(%)

    Controla la opacidad de la imagen. 1 es el valor por defecto y 0 es transparente

  • saturate(%)

    Controla la saturación de color de la imagen. 1 es el valor por defecto, 0 es totalmente desaturada y por encima de 1 se sobresatura

  • sepia(%)

    Aplica un tono sepia a la imagen. 0 es el valor por defecto y 1 es totalmente sepia.

  • url()

    busca información, este se aplica en svg.
