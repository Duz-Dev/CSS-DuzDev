Flexbox

Fundamentos

Introducción

Flexbox es un modulo de css que permite que las cajas sea flexibles. Esta es un valor de la propiedad display al igual como lo es inline y block.

Existe displayflexyinline-flexpara que el contenedor sea de bloque o de linea respectivamente.

Se necesita un contenedor (flex-container) y al menos un hijo (flex-item). Con ello decimos que si el container no tiene hijos aunque sea uno no serviría de nada aplicar flex, ya que no tendría caso usarlo, en cualquier caso seria mas util usar block directamente.

Las cajas flexibles son los items, el contenedor es el contexto al que pertenecen los items. No porque a un contenedor le demos display flex significa que lo convierta en una caja flexible, lo que se hace flexible son sus hijos, si queremos hacer flexible por alguna razón al contenedor tendríamos que ponerle flex a el contenedor de dicho contenedor.

Y solo los hijos directos de el container serán flexibles, así que si dentro de uno de los hijos este tiene hijos entonces no le afectara el display flex.

El texto se considera un hijo del contenedor por lo que el texto se convierte en una caja flexible, y los after y before igual.

Cuando utilizamos display flex o inline-flex suceden cosas de tras de css como se menciona a continuación:

Se generan dos ejes para colocar y alinear los elementos

Por defecto el eje principal va en horizontal (de izquierda a derecha) y el eje secundario que da en vertical (de arriba hacia abajo)

se puede intercambiar los ejes y la dirección. Por ejemplo pódemos decir que el eje principal sea el vertical y que vaya de abajo hacia arriba.

Estos ejes reciben un nombre y se utilizan en la terminología a la hora de trabajar con Flexbox.

flex-direction

Con esta propiedad podemos determinar que dirección tomara el Main axis. Sus valores son:

  • row

    Es la dirección por defecto que toma el main axis, el que va de horizontal que inicia de derecha y termina en la izquierda.

  • row-reverse

    Es la dirección opuesta entre el inicio y el final y se mantiene el eje horizontal.

  • column

    Es la dirección que toma el main axis en vertical, y el cross axis tomaría el eje horizontal, y los flex-item irían de arriba hacia abajo.

  • column-reverse

    Es la dirección que toma el main axis en vertical, y el cross axis tomaría el eje horizontal, y los flex-item irían de abajo hacia arriba.

flex-wrap

Con esta propiedad podemos determinar que dirección tomaran los elementos cuando no quepan en s contenedor. Sus valores son:

  • nowrap

    Este es le valor por defecto que tiene flex, este indica que sus flex-items se acomoden solo en una linea, por lo que el ancho de dichos items se vera afectado para forzar el acomodo de los hijos.

  • wrap

    este indica que sus flex-items se acomoden dependiendo del ancho, asi que si el ancho de uno de los items topa con el ancho de el otro este se desplazara hacia abajo ya que sabe que ya no hay espacio en la primera linea (es como usar display block pero este varia).

  • wrap-reverse

    Es lo mismo que wrap solo que en lugar de que los items fluyan hacia abajo lo aran hacia arriba. Este valor en particular casi no siempre se utiliza.
