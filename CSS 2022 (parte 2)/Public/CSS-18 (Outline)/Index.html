Propiedad Outline

outline

¿Que es?

Como vemos en el titulo outline es una propiedad de css, esta se comporta casi igual como border(ver modelo de cajas de CSS-Básico), pero este no afecta a la caja, es decir, da igual las medidas que tenga este no alterara el layout. Este es como si fuera un borde, pero como no forma parte del modelo de cajas no empuja el contenido, a diferencia de border que si lo hace.

Sintaxis

Igual que border, pero con el plus que podemos especificarle que se salga de la caja con la propiedad outline-offset. Esta misma propiedad si le aplicamos valores positivos se ira saliendo de la caja, pero si aplicamos valores negativos se ira metiendo dentro de esta. La ventaja de este es que podemos crear un marco pero que no empuje el contenido o utilizar un pseudo-elemento.
