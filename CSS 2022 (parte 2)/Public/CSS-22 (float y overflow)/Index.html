Overflow y float

Overflow

Propiedad:

Esta propiedad va de la mano de el modelo de caja se podría decir, ya que esta sirve para indicar que todo lo que se desborde (salga) de su contenedor se oculte o se creé un scroll, esto ultimo ya dependerá de el valor que le asignemos.

Sintaxis

Se escribe comooverflowy esta se puede aplicar a todos los contenedores, y sus valores son:

  • auto

    Crea un scroll en la parte derecha de la caja para poder desplazarnos dentro de su contenido

  • scroll

    lo mismo que que auto solo que muestra igualmente el scroll horizontal aunque no haya elementos que se desborden hacia dicha dirección.

  • hidden

    Todos los elementos que se salgan del contenedor no se visualizaran.

Float

Propiedad

Esta propiedad se usaba antes para todo ya que era la unica manera de poder orientar los elementos de una página web. Actualmente existe el modulo de flexbox y grid que son con lo que Actualmente se maqueta.

Float aun y asi sigue siendo util pero con lo anterior dicho este ya no relevante, es, mas este no se deberia de usar ya que este genera muchos problemas que se pueden evitar sabiendo los dos modulos antes mencionados.

Pero no todo es malo con float, con el podemos mandar a flotar a elementos bloque y con ello poder generar efectos curiosos que solo es posible con float, por ejemplo mandar a flotar una imagen a la derecha para que el texto que tenia debajo se recorra a la izquierda y a si vez si dicho texto sobre pasa el tamaño del texto pues texto tomara el tamaño del 100% nuevamente ahora que la imagen no le estorba. Como vemos se auto calcula y esto solo sucede con float.

Float es un tema muy extenso debido a su uso anteriormente, y como este tiene varios fallos pues era comun recurrir a trucos para repararlos. (como dato este sitio usa float para el acomodo de el layout .__.)
