Sintaxis y Selectores

Sintaxis

Estructura

At-rules

a la arroba @ en ingles se escribe at por lo que sabemos que at-rules significan arroba de reglas, estas son reglas especiales que hacen referencia a distintas reglas css pre-programadas en el lenguajes. Estas vienen siendo @media{}, @font-face{}, @keyframes{} y @import().

  • @media{}

    Este sirve para especificar que clases se aplicaran dependiendo de la medida que le asignemos las llaves.

  • @font-face{}

    Este sirve para importar diferentes familias tipográficas.

  • @keyframes{}

    Este sirve para especificar los efectos que se aplicaran a una animación con respecto el intervalo o el fragmento clave.

  • @import()

    Este sirve para importar hojas de estilo en el propio css, esto no es recomendable ya que si dicha hora no se encuentra ningún estilo posterior a ello se aplicara.

media@{720px} { selector { propiedad: valor ; } }

Selectores

Tipos de selectores

Estos selectores como ya vimos son lo que usamos para hacer referencia a que elementos del html le vamos aplicar estilos, pues bien estos selectores son variados y se pueden clasificar en cuatro tipos de selectores.

  • Selectores básicos

    Estos vendrían siendo los selectores de etiqueta y de atributo.

  • Selectores conminadores

    Son los símbolos que hacen referencia a cierta ligadura con los otros elementos. Por ejemplo el simbolo de + hace alusión de que se seleccionara el hermano que este debajo del primer selector.

  • Selectores de pseudo-elementos

    Son selectores que se usan para manipula elementos creados a partir de css o elementos ya muy específicos de html, y se escriben con cuatro puntos ::.

  • Selectores de pseudo-clases

    Estos se aplican a casi todos los elementos y varian dependiendo de su función las cuales son de mucha ayuda. estas se escriben con dos puntos :.

!Con los selectores conminadores seria mas o menos asi:
.soy-un-h1 + h2 { color:red; } /*Con el + indicamos que seleccione al a los h2 que este adyacente de el .soy-un-h1, vaya selección al h2 hermano mas cercano del .soy-un-h1.*/ h1 ~ h2 { color:green; } /*seleccionara a los h1 hermanos que estén debajo de el, a todos.*/
!Los selectores dependientes, de papá a hijo
.div > span { color:lime; } /*Con el mayor que [>] le indicamos que seleccione al hijo directo, en este caso en concreto le indicamos que si el hijo mas cercano al div es un span seleccionalo, de lo contrario no apliques nada. Siendo mas claros, con > le indicamos que seleccione a los hijos que no estén dentro de otras etiquetas, con ello especificamos que si hay varios span pero dentro de un div (el papá seria un div que tiene un hijo div que tiene varios span) estos no se seleccionarían.*/
