Introducción a CSS

¿Qué es CSS?

  • Css no es un lenguaje de programación, aunque tiene variables y funciónes. No podemos crear funciones ya que están integradas.

  • Es el único lenguaje de diseño que existe. Css es el único lenguaje para darle diseño a la web.

  • CSS significa "Cascade Style stylesheet" que es en español se traduce como Hojas de estilo en cascada.

Historia de CSS

Puntos importantes de la historia de CSS:

  • 1970- Se crea un lenguaje para darle estilo a los documentos de SMGL. SMGL es lo que vino antes de HTML.

  • 1994/1995- Se unió SSP Y CHSS para definir el lenguaje CSS. Estos dos lenguajes eran propuestas para darle estilos a la web.

  • 1996- La W3C publica el estándar CSS1.W3C son las siglas de World Wide Web Consortium, y es una comunidad internacional donde los estados miembros trabajan para poder desarrollar estándares para el desarrollo web y asi ayudar a un mejor desarrollo de el Internet a nivel mundial.

  • 1998- La W3C publica estándar para CSS2.

  • 2009- La W3C publica estándar para CSS2.1. Este no llego a ser CSS3 porque lo único que hacia era añadir unos estilos mas.

  • 2011:

    • A partir de CSS3 la W3C empieza a lanzar Snapshots en vez de estándares.

    • Con la llegada de CSS3 el lenguaje se fragmenta y se divide en distintos módulos y cada uno tiene su propio estándar o recomendación oficial.

    • Cada módulo se divide en sus propios niveles y sus grupos de trabajo por lo que la 3WC no habrá CSS4, si no que cada modulo podrá llegar a nivel 1,2,3,4 etc. Si un modulo no se utíliza o surge algo que lo sustituya, este queda obsoleto y se elimina.

Gracias a CSS3

  • La llegada de CSS3 fue una revolución para el mundo web.

  • Nació el RWD (Responsive Web desing). Hasta CSS3 no hacia falta adaptar la web ya que no teníamos internet en los móviles, entonces con la llegada de los smartphones.

  • Podemos elegir estilos distintos para cada tamaño de pantalla e incluso para impresoras.

  • Llego flexbox y grid que ha sido un gran avance para el diseño web. Gracias a ellos podemos crear páginas que antes de estos dos módulos eran imposibles.

  • Llegan las animaciones y las transiciones.
