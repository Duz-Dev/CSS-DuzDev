Font-face

@font-face

¿Que es?

@font-facees la at-rule que podemos usar para importar familias de fuentes (y dentro de esta están las fuentes que usamos) para poder mostrarlas en nuestra página web. Es importante recordar que al momento de usar font-family le decimos al navegador que busque tal fuente en el equipo y si esta no esta pondrá la que le hayamos concatenado. Con ello esta el problema de que no todos los dispositivos tienen las mismas fuentes instaladas por lo que nosotros para evitar esto usamos esta regla para importar nuestras fuentes en lugar de que utilicemos las instaladas en el equipo.

¿Comó las importamos?

Simplemente es conseguir la fuente, esta la podemos insertar por medio de un link que pegaríamos en el html, un import de css o descargándolo.

Sintaxis:

!Sintaxis
@font-face( src: url('Dirección de la feunte'); font-family: Cualquier nombre; }

Usar fuentes descargadas

Para descargar las fuentes y luego utilizar podemos usar las páginas de dafont, Google fonts o 1001fonts por dar algunos ejemplos, después dentro de la carpeta donde guardamos todo lo img música y demás (que generalmente se llama assets) las guardamos allí, y aquí podemos partir de dos métodos: El primero seria crear un css donde pongamos los @font-face de todas las fuentes que queramos y ya después usar el important dentro de nuestra hoja de estilos principal y llamar a la hoja de las fuentes con un @import, y el otro es directamente importar las fuentes al css. Evidentemente cada una tiene sus ventajas y desventajas que dependiendo del proyecto ya dependería si usar el primer método o el segundo.
